# Thor
C Language Framework

Idealizado em 2006 para agilizar o desenvolvimento de aplicações em C e criar funções aumentando segurança e padronização de código.

Inicialmente baseado nos ensinamentos encontrados nos livros "Advanced Programming in the UNIX Environment", "UNIX Networking Programming" e "Programando em C/C++ - A Bíblia" e posteriormente expandido e atualizado com necessidades encontradas ao longo do tempo.

Tendo sido utilizado para uso pessoal desde sua criação, em 2026 o framework foi revisado e disponibilizado como público no Github.
